What I've built

PROJECT_01
GraphRAG — Knowledge Graph Augmented Retrieval
Production-grade RAG pipeline combining Qdrant vector search with Neo4j knowledge graph memory for context-aware AI responses. LangGraph handles stateful agent orchestration with checkpointing and human-in-the-loop interruptions. LangFuse provides full LLM observability. MinIO manages document storage. All 5 services run in a single docker-compose environment with Nginx reverse proxy.

Autonomous AI Agent System — MCP Servers & Tool Binding
Multi-agent orchestration system using LangGraph with tool binding, API calling, and MCP server integration. Implements self-hosted LLMs (Llama-3, Gemma) with guardrails, PII detection, and prompt filtering. Features LLM-as-a-Judge evaluation for response quality, Perplexity Sonar API for real-time retrieval, model fine-tuning, and both autonomous and controlled workflow strategies.
